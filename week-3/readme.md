# 🎮 Game Center (HTML Mini Games)

This project is a simple web-based game collection built with **HTML, CSS, and JavaScript**.

You can play three classic mini games from a single menu page.

---

## 🚀 Games Included

### 🏓 Ping Pong Game
- Classic paddle game
- Control using keyboard
- Try to keep the ball in play!

### 🧩 Maze Game
- Navigate through a maze
- Reach the goal to win
- Use arrow keys to move

### 🧱 Brick Breaker Game
- Break all bricks with the ball
- Control the paddle using arrow keys
- Track score and lives

---

## 📂 Project Structure
/project-folder
│── index.html # Main menu
│── ping-pong-game.html # Ping Pong game
│── maze-game.html # Maze game
│── brick-breaker-game.html # Brick Breaker game
│── README.md

## ▶️ How to Run

1. Download or clone this repository
2. Open `index.html` in your browser
3. Select a game from the menu
